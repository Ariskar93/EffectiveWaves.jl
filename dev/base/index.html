Types of waves

There are currently three main types used.

EffectiveWaves.EffectiveWave — Type.

A type for the effective wave ansatz.

source
EffectiveWaves.effective_waves — Function.

Calculates the effective wavenumbers and return Vector{EffectiveWave}.

source
EffectiveWaves.AverageWave — Type.

A type for the ensemble average scattering coefficients. Here they are discretised in terms of the depth x of the halfspace

source
EffectiveWaves.average_error — Function.

Approximates the error in AverageWave.amplitudes due to the mesh AverageWave.x.

source
EffectiveWaves.x_mesh — Function.

Returns x the mesh used to discretise the integral equations.

source
EffectiveWaves.MatchWave — Type.

A type for matched waves.

source
EffectiveWaves.match_error — Function.

Calculates the difference between the match of MatchWave.effectivewaves and MatchWave.averagewave. This can be used as a proxi for convergence.

source
EffectiveWaves.x_mesh_match — Function.

Returns (x,L), where x[L:end] is the mesh used to match with wave_effs.

source
